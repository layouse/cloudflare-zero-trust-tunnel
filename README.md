# cloudflare-zero-trust-tunnel
Debian 12/13 üzerinde Apache sanal konakları ile Cloudflare Zero Trust Tunnel kurulumu, çoklu subdomain yönetimi ve karşılaşılan hataların çözümlerini içeren kapsamlı rehber. - This comprehensive guide covers setting up Cloudflare Zero Trust Tunnel with Apache virtual hosts on Debian 12/13, managing multiple subdomains, and resolving common errors.
